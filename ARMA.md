$$\gamma(\tau) = \text{cov}(X_t, X_{t+\tau}) = 
\begin{cases} 
(1 + \theta_1^2 + \theta_2^2)\sigma^2 & \text{for } \tau = 0, \\
(\theta_1 + \theta_1\theta_2)\sigma^2 & \text{for } \tau = \pm1, \\
\theta_2\sigma^2 & \text{for } \tau = \pm2, \\
0 & \text{for } |\tau| > 2.
\end{cases}$$
